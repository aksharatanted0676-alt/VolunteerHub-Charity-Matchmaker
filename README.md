# VolunteerHub: Charity Matchmaker

VolunteerHub is a web-based platform that connects volunteers with local charities based on their skills and location.

## Technologies Used
HTML
CSS
JavaScript

## Features
- Volunteer registration
- Opportunity listings
- Skill-based matching
- Location-based search
- Volunteer dashboard
